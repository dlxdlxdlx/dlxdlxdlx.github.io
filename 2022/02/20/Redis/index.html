Redis

Created2022-02-20|Updated2022-03-07|Computer
|Word count:522|Reading time:1min|Post View:

简介:
Redis是一个高性能的key-value数据库，它提供了一个高性能的数据结构，可以提供高速的查询和更新，并且支持高并发的处理。

  • 支持数据持久化
  • 不仅支持key-value 数据,还提供list,ste,zset,hash等数据结构存储.
  • 支持数据备份.

Redis 优势

  • 多数据类型 支持二进制strings,Lists,Sets,Sorted Sets,Hashes,和HyperLogLog类型的数据操作
  • 高性能
  • 原子性: 对于所有的操作,Redis都是原子性的,即操作是一个完整的单元.

安装:

基础数据类型

strings

通过 SDS 存储(通过预分配冗余空间减少内存频繁分配.)

常见应用场景:

  • 缓存: 利用redis作为缓存,其他数据库作为存储层,利用redis的高并发特点加快系统读写速度以及降低后端数据库压力.
  • 计数器: 作为系统实时计数器,加速实现计数和查询功能.
  • 共享Session: 利用Session对用户数据进行集中管理,这样在用户刷新页面时可以实现user session的快速获取.

Hash:

由字符型field与value组成的映射表结构.

常用操作

command comment
HDEL key field 删除一个或者多个hash 字段
HEXISTS key field 判断hash 字段是否存在
HGET key field 获取key 关联的哈希字段值
HGETALL key 获取key 关联的所有哈希字段
HINCRBY key field increment 对key关联的字段做增量运算
HINCRBYFLOAT key field increment 对key关联的字段做浮点增量运算
HKEYS key 获取key关联的所有字段
HLEN key 获取key关联的哈希字段个数
HMSET key field1 value1 […] 设置多个字段
