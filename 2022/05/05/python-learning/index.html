数据类型

python 包含5个基本数据类型

  1. Number(数字)
  2. String(字符串)
  3. List(列表)
  4. Tuple(元组)
  5. Dictionary(字典)
  6. Set(集合)

其中
Number , String , Tuple 是不可变的
List , Dictionary , Set 是可变的

Number

  • int
  • float
  • complex
  • bool (python3 中bool 是int的子类,所以可以与int进行运算)

数值运算

      • 省略

/ 表示除法, //表示整数除法

String

python中 支持以下两种方式,遇到特殊字符需要使用反斜杠进行转义(如果不想转义需要使用r)

  • String是不可变类型 ,类似a[0]=’h’是不被允许的

List

  • 最常用的数据类型
  • 一个list中可以存储多个不同种类的数据类型
  • 可以使用列表截断对list进行访问

Tuple

使用 圆括号包裹元素,不可变类型,可以用来定义函数的多个返回值类型
可以使用+ 进行拼接

set

创建空集合只能使用Set()

Set 可以使用运算符进行操作

Dictionary

使用{}包裹元素,可以使用key:value的形式来存储数据
